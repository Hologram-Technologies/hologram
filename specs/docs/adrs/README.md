## Process

1. Copy the ADR template from `hologram-architecture` (or use `holoarch onboard`).
2. Fill in Context, Decision, Consequences, Alternatives.
3. Propose for review; mark Accepted or Rejected.
4. Link the ADR from the relevant doc in `specs/docs/`.
