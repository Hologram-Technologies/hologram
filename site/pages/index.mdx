# Hologram

**O(1) neural network inference via precomputed lookup tables and KV-dispatch.**

Hologram is a v2 rewrite targeting extreme inference speed by replacing floating-point arithmetic with integer LUT lookups and key-value dispatch, built on [`uor-foundation`](https://github.com/UOR-Foundation/uor-foundation) v3.5.0.

## Core Idea

Instead of computing `f(x)` at runtime, every activation and arithmetic operation is resolved as a table lookup:

```
result = LUT[encoded_input]   // O(1), integer index
```

Weights are quantized to integer ring elements. Matrix multiplication becomes a sum of LUT lookups (`LUT-GEMM`), eliminating floating-point entirely on hot paths.

## Crates

| Crate | Role |
|-------|------|
| [hologram-core](/crates/core) | LUT tables, ring algebra, encoding, views (`no_std`) |
| [hologram-graph](/crates/graph) | Expression graph, fusion, parallel scheduling |
| [hologram-archive](/crates/archive) | `.holo` file format, rkyv zero-copy, mmap |
| [hologram-exec](/crates/exec) | KV executor, buffer arena, parallel level execution |
| [hologram-compiler](/crates/compiler) | Graph → schedule compilation |
| [hologram-async](/crates/async) | Tokio async execution + streaming |
| [hologram-ffi](/crates/ffi) | C ABI + WASM bindings |

## Quick Start

```bash
git clone https://github.com/UOR-Foundation/hologram
cd hologram
just ci        # fmt + clippy + test
cargo run --example calculator
```
