Hologram is a v2 rewrite targeting extreme inference speed by replacing floating-point arithmetic with integer LUT lookups and key-value dispatch, built on [`uor-foundation`](https://github.com/UOR-Foundation/uor-foundation) v3.5.0.

## Core Idea

Instead of computing `f(x)` at runtime, every activation and arithmetic operation is resolved as a table lookup:

```
result = LUT[encoded_input]   // O(1), integer index
```

Weights are quantized to integer ring elements. Matrix multiplication becomes a sum of LUT lookups (`LUT-GEMM`), eliminating floating-point entirely on hot paths.
